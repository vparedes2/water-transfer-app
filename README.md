# Water Transfer Track

Sistema de control de activos para water transfer.

## Uso
1. Abre index.html en tu navegador
2. Se cargarán 400+ activos de ejemplo automáticamente
3. Usa el mapa para ver las operaciones en Añelo

## Módulos
- Dashboard: Resumen de activos
- Mapa: Operaciones con trazas GPS
- Carga Campo: Registro de montajes/desmontajes
- Inventario: Todos los activos
- Reportes: PDFs y exportación de datos

## Datos
Los datos se guardan en LocalStorage del navegador.
